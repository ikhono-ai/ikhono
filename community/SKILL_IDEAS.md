# Skill Ideas

Looking for a skill to build? Here are ideas the community would love to see. Pick one, build it, and publish it on iKhono!

## Code Quality

- **Clean Code Reviewer** — Check code against clean code principles (SOLID, DRY, KISS)
- **Refactoring Assistant** — Identify and apply refactoring patterns (extract method, rename, simplify)
- **Complexity Analyzer** — Flag functions with high cyclomatic complexity and suggest simplifications
- **Dead Code Finder** — Identify unused exports, functions, and variables

## Security

- **OWASP Top 10 Reviewer** — Check code for the 10 most critical web security risks
- **Dependency Auditor** — Review package dependencies for known vulnerabilities
- **Auth Flow Reviewer** — Validate authentication and authorization implementations
- **Secret Scanner** — Find hardcoded secrets, API keys, and credentials in code

## Testing

- **Test Coverage Analyzer** — Identify untested code paths and suggest test cases
- **E2E Test Writer** — Generate end-to-end tests for web applications
- **API Test Generator** — Create integration tests for REST/GraphQL APIs
- **Snapshot Test Updater** — Update snapshot tests with clear explanations of changes

## Documentation

- **README Generator** — Create comprehensive READMEs from project structure and code
- **JSDoc/TSDoc Writer** — Generate type-safe documentation comments
- **Architecture Documenter** — Create architecture decision records (ADRs)
- **Migration Guide Writer** — Generate upgrade guides between versions

## DevOps & Infrastructure

- **Dockerfile Reviewer** — Optimize Dockerfiles for size, security, and caching
- **CI/CD Pipeline Reviewer** — Check GitHub Actions / GitLab CI configs for best practices
- **Terraform Reviewer** — Validate infrastructure-as-code for security and cost
- **Kubernetes Manifest Checker** — Review k8s configs for common misconfigurations

## Language-Specific

- **TypeScript Strict Mode Helper** — Migrate JavaScript to strict TypeScript
- **Python Type Hints Writer** — Add type annotations to Python code
- **Go Error Handling Reviewer** — Check Go code for proper error handling patterns
- **Rust Lifetime Helper** — Explain and fix lifetime issues in Rust code

## Database

- **SQL Query Optimizer** — Review and optimize SQL queries for performance
- **Schema Reviewer** — Check database schemas for normalization, indexing, and naming
- **Migration Writer** — Generate database migrations from schema changes

## Git & Collaboration

- **Commit Message Writer** — Generate conventional commit messages from diffs
- **PR Description Generator** — Create structured PR descriptions with context and test plans
- **Changelog Generator** — Generate changelogs from commit history
