# Skill Ideas

Looking for a skill to build? Here are ideas the community would love to see. Pick one, build it, and publish it on iKhono!

## Writing & Communication

- **Email Composer** — Draft professional emails for any context (follow-ups, introductions, complaints)
- **Resume Builder** — Tailor resumes and cover letters for specific job descriptions
- **Blog Post Writer** — Structure and draft blog posts with SEO-friendly headings
- **Proofreader** — Check grammar, tone, and clarity for any type of writing
- **Meeting Notes Summarizer** — Turn raw meeting notes into structured summaries with action items

## Finance & Business

- **Budget Planner** — Create personal or business budgets from income and expense data
- **Invoice Generator** — Draft professional invoices with line items and payment terms
- **Financial Report Analyst** — Summarize and explain financial statements and reports
- **Tax Prep Helper** — Organize documents and identify deductions for tax filing
- **Business Plan Writer** — Structure and draft business plans section by section

## Learning & Education

- **Study Guide Creator** — Generate study guides and flashcards from any topic or material
- **Concept Explainer** — Break down complex topics into simple, layered explanations
- **Quiz Generator** — Create quizzes and practice tests from any subject matter
- **Language Practice Partner** — Conversational exercises and grammar drills for language learners
- **Research Assistant** — Summarize papers, find key arguments, and organize references

## Design & CSS

- **CSS Layout Helper** — Generate flexbox, grid, and responsive layouts from descriptions
- **Color Palette Generator** — Create harmonious color schemes for any project or brand
- **UI Component Builder** — Scaffold styled UI components (buttons, cards, forms, navbars)
- **Animation Writer** — Create CSS/JS animations and transitions from plain descriptions
- **Design System Reviewer** — Check UI consistency against design tokens and guidelines

## Productivity & Organization

- **Project Planner** — Break down goals into tasks, milestones, and timelines
- **Decision Matrix Builder** — Structure pros/cons and weighted criteria for tough decisions
- **Habit Tracker Setup** — Design personalized habit tracking systems and routines
- **Travel Planner** — Build itineraries with logistics, budgets, and local recommendations
- **Event Planner** — Organize events with checklists, timelines, and vendor coordination

## Data & Analysis

- **CSV Analyzer** — Explore, clean, and summarize data from CSV files
- **Survey Builder** — Design effective survey questions and analyze response data
- **Chart Recommender** — Suggest the best visualization type for your data and generate it
- **Spreadsheet Formula Helper** — Write and explain complex Excel/Sheets formulas

## Code Quality

- **Clean Code Reviewer** — Check code against clean code principles (SOLID, DRY, KISS)
- **Refactoring Assistant** — Identify and apply refactoring patterns
- **Complexity Analyzer** — Flag high-complexity functions and suggest simplifications
- **Dead Code Finder** — Identify unused exports, functions, and variables

## Security

- **OWASP Top 10 Reviewer** — Check code for the 10 most critical web security risks
- **Dependency Auditor** — Review package dependencies for known vulnerabilities
- **Secret Scanner** — Find hardcoded secrets, API keys, and credentials in code

## Testing

- **Test Coverage Analyzer** — Identify untested code paths and suggest test cases
- **E2E Test Writer** — Generate end-to-end tests for web applications
- **API Test Generator** — Create integration tests for REST/GraphQL APIs

## Documentation

- **README Generator** — Create comprehensive READMEs from project structure and code
- **Architecture Documenter** — Create architecture decision records (ADRs)
- **Migration Guide Writer** — Generate upgrade guides between versions

## DevOps & Infrastructure

- **Dockerfile Reviewer** — Optimize Dockerfiles for size, security, and caching
- **CI/CD Pipeline Reviewer** — Check GitHub Actions / GitLab CI configs for best practices
- **Kubernetes Manifest Checker** — Review k8s configs for common misconfigurations

## Content & Marketing

- **Social Media Post Writer** — Draft platform-specific posts (LinkedIn, Twitter/X, Instagram)
- **Product Description Writer** — Create compelling product descriptions for e-commerce
- **SEO Content Optimizer** — Improve content for search engine visibility
- **Newsletter Writer** — Draft engaging email newsletters with clear CTAs

## Health & Wellness

- **Meal Planner** — Create weekly meal plans based on dietary goals and preferences
- **Workout Routine Builder** — Design exercise routines tailored to fitness goals
- **Mindfulness Guide** — Generate guided meditation scripts and breathing exercises

## Legal & Compliance

- **Contract Reviewer** — Summarize key terms and flag potential issues in contracts
- **Privacy Policy Generator** — Draft GDPR/CCPA-compliant privacy policies
- **Terms of Service Writer** — Create ToS documents tailored to your product type
